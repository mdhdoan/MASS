 1. Keywords: Juvenile salmonids density estimation, BACI design, stream reach sampling, matched control reaches
2. Abstract: This method discusses the importance of estimating the density of juvenile salmonids at project sites to monitor changes in abundance over time. A Before-After-Control-Impact (BACI) experimental design is employed, where the impacted stream reach is compared to a control reach of similar characteristics. This approach allows for a more accurate assessment of the effects of the project on juvenile salmonid populations.
3. Description: The text describes a method used to estimate the density of juvenile salmonids in a project-impacted stream reach, with the aim of understanding changes in abundance over time. To improve accuracy and isolate project effects, this method utilizes a BACI experimental design, which involves comparing the impacted stream reach to a control reach that shares similar length and size characteristics on the same stream. This comparison enables researchers to better understand how the project influences juvenile salmonid populations while accounting for natural fluctuations in abundance.
4. Target: The primary focus of this method is estimating the density and abundance changes of juvenile salmonids in a project-impacted stream reach, employing a BACI experimental design to increase accuracy and reliability.
5. Constraints: This method's constraints include the availability of suitable control reaches on the same stream with comparable length and size characteristics to the impacted reach, as well as consistent monitoring efforts to ensure accurate density estimations over time.
